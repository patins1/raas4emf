Following steps must be done to start the server:
1. Execute:	C:\bim\NOLServer\update_and_restart.bat



Following steps must be done to access the object Library in a browser:
1. Visit folling site in Firefox or Google Chrome:	http://localhost:80/raascms/nol



Following steps must be done to use the local server for Revit and Archicad:
1. Read the instructions at C:\bim\NOLClient\readme.txt
2. In file C:\bim\NOLClient\config.ini, change the following key value pair:
   BIMUIURL = http://localhost:80/raascms/nol
